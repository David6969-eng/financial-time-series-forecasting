/ Methodology

Short-Time Fourier Transform

Financial time series are non-stationary — their statistical properties change over time. STFT captures both time and frequency information simultaneously.

STFT Definition
STFT(t, f) = ∫ X(τ) · w(τ−t) · e−j2πfτ dτ
Spectrogram (Energy)
S(t, f) = | STFT(t, f) |²
Sliding Window Segments
X1:L, X1+H:L+H, X1+2H:L+2H, ...
  • LWindow Length — number of samples per segment
  • HHop Size — step by which the window shifts
  • L − HOverlap between consecutive windows
  • S ∈ ℝT×FSpectrogram treated as a 2D image
↑ Larger WindowBetter frequency resolution, weaker time resolution
↓ Smaller WindowBetter time resolution, weaker frequency resolution
Live Spectrogram Simulation
← Time →↑ Frequency ↑
Spectrogram Interpretation
  • Low FreqLong-term trends in financial data
  • High FreqShort-term fluctuations & volatility
  • IntensitySignal energy at (time, frequency)
  • FormatTreated as a 2D image for CNN input
